LabQuiz

O LabQuiz é um quiz de 8 perguntas para desafiar seus conhecimentos em HTML5 e CSS.

Obs: as questões 6,7,8 são de multipla escolha, e todas as questões abertas são expansíveis arrastando no canto inferior direto da caixa de texto
























1° Como se define um estilo inline em HTML?




2° Qual é a tag usada para criar uma lista ordenada em HTML?




3° Qual propriedade CSS é usada para alterar a cor do texto?




4° Como se referencia um arquivo CSS externo em um documento HTML?




5° Font-size é a propriedade CSS usada para alterar o tamanho da fonte de um texto?




6° Qual é a propriedade CSS usada para definir a largura de um elemento?




7° Como se cria um comentário em CSS?




8° Qual é a propriedade CSS usada para definir a altura de um elemento?
